# tock

<img src="img/tock.svg" align="right" width="150">

**The fast terminal calendar. Written in Rust.**

![Rust](https://img.shields.io/badge/language-Rust-f74c00) ![License](https://img.shields.io/badge/license-Unlicense-green) ![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20macOS-blue) ![Status](https://img.shields.io/badge/status-planned-yellow) ![Stay Amazing](https://img.shields.io/badge/Stay-Amazing-important)

Terminal calendar with day, week, and month views, event management, and astrological data. Built on Crust. Feature clone of [Timely](https://github.com/isene/Timely) rewritten in Rust for speed and single-binary distribution.

## Status

**Planned.** This project is part of the Rust Terminal Suite.
